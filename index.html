Introduction to C Programming

C is a general-purpose, structured programming language developed by Dennis Ritchie at Bell Labs.

It is widely used for:

  • Operating systems (e.g., UNIX components)
  • Embedded systems
  • Compilers and interpreters
  • Performance-critical applications

Why we study C:

  • Helps understand how computers manage memory
  • Forms the base for languages like C++, Java, etc.
  • Fast, portable, and close to hardware

Structure of a C Program

A C program is typically divided into these sections:
  • Documentation Section – Comments about the program
  • Link Section – Header file inclusion using 
    #include
  • Definition Section – Macros using 
    #define
  • main() Function – Starting point of program
  • Subprogram Section – User-defined functions

Code Example:


#include<stdio.h>       // Link section
#define PI 3.14              // Definition section
int globalVar = 10;          // Global declaration
void display();              // Function declaration
int main() {                 // main() function
    printf("Hello C!\n");
    display();
    return 0;
}
void display() {             // Subprogram section
    printf("Global = %d\n", globalVar);
}

  • Digits: 0–9
  • White Spaces: space, tab, newline
  • Special characters: $, @, #, %, ^, *, ?, +, -

Tokens in C

Tokens are the smallest meaningful units in a C program.

Types of tokens:

  • Keywords (e.g., int, if, return)
  • Identifiers (names given to variables, functions)
  • Constants (e.g., 10, 3.14, 'A')
  • Strings (e.g., "Hello")
  • Operators (e.g., +, -, ==)
  • Special symbols (e.g., ;, {, })

Code Example


int total = 10;

//Tokens here:
 int (keyword),
 total (identifier),
 = (operator),
 10 (constant),
 ; (special symbol)

Identifiers

Identifiers are names given to variables, functions, and other entities in a program.

Rules for identifiers:

  • Must start with a letter or underscore
  • Can contain letters, digits, and underscores
  • Are case-sensitive (e.g., total and Total are different)
  • Cannot be keywords (e.g., int, if, return)

Code Example:


							int total = 10;
							int Total = 20;
							int _total = 30;
							int 1total = 40; // Invalid
							int int = 50; // Invalid

Looping Statements

Loops repeat a block of code as long as a condition is true.

Types

Syntax

Syntax

Syntax

    // statements

Example

goto, break and continue

break

        break;          // exits the loop
    printf("%d ", i);
}

continue


for (int i = 1; i <= 5; i++) {
    if (i == 3)
        continue;       // skip rest for i == 3
    printf("%d ", i);

goto


int x = 0;
start:
printf("x = %d\n", x);
x++;
if (x < 3)
    goto start;         // jumps to label

(Use goto rarely; it makes code hard to read.)

Swap Two Numbers Without Using a Third Variable

Algorithm (Concept)

Read two numbers A and B

A = A + B

B = A - B

A = A - B

Display A and B

Code:


#include <stdio.h>

int main() {
    int a, b;
    printf("Enter two numbers: ");
    scanf("%d%d", &a, &b);

    printf("Before swap: a = %d, b = %d\n", a, b);

    a = a + b;
    b = a - b;
    a = a - b;

    printf("After swap: a = %d, b = %d\n", a, b);
    return 0;
}

Linear Search in an Array

Check each element of the array sequentially until the element is found or the array ends.

Code:


#include <stdio.h>

int main() {
    int a[5] = {4, 7, 1, 9, 3};
    int key, i, found = 0;

    printf("Enter element to search: ");
    scanf("%d", &key);

    for (i = 0; i < 5; i++) {
        if (a[i] == key) {
            printf("Element found at index %d\n", i);
            found = 1;
            break;
        }
    }

    if (!found)
        printf("Element not found\n");

    return 0;
}

Bubble Sort

Concept

Repeatedly compare adjacent elements

Swap them if they are in the wrong order

Largest element “bubbles” to the end after each pass

Code:


#include <stdio.h>

int main() {
    int a[5] = {5, 1, 4, 2, 8};
    int i, j, temp;

    for (i = 0; i < 5 - 1; i++) {
        for (j = 0; j < 5 - 1 - i; j++) {
            if (a[j] > a[j+1]) {
                temp = a[j];
                a[j] = a[j+1];
                a[j+1] = temp;
            }
        }
    }

    printf("Sorted array: ");
    for (i = 0; i < 5; i++)
        printf("%d .", a[i]);

    return 0;
}

Program to Find A² + B for 2×2 Matrices

Concept

First compute A² = A × A

Then compute R = A² + B

Code:


#include <stdio.h>

int main() {
    int A[2][2], B[2][2], temp[2][2], R[2][2];
    int i, j, k;

    printf("Enter elements of matrix A (2x2):\n");
    for (i = 0; i < 2; i++)
        for (j = 0; j < 2; j++)
            scanf("%d", &A[i][j]);

    printf("Enter elements of matrix B (2x2):\n");
    for (i = 0; i < 2; i++)
        for (j = 0; j < 2; j++)
            scanf("%d", &B[i][j]);

    // Compute temp = A * A (A^2)
    for (i = 0; i < 2; i++) {
        for (j = 0; j < 2; j++) {
            temp[i][j] = 0;
            for (k = 0; k < 2; k++)
                temp[i][j] += A[i][k] * A[k][j];
        }
    }

    // Compute R = A^2 + B
    for (i = 0; i < 2; i++)
        for (j = 0; j < 2; j++)
            R[i][j] = temp[i][j] + B[i][j];

    printf("Result (A^2 + B):\n");
    for (i = 0; i < 2; i++) {
        for (j = 0; j < 2; j++)
            printf("%d ", R[i][j]);
        printf("\n");
    }

    return 0;
}

Transpose of a Matrix

Concept

In the transpose of a matrix:

Rows become columns

Columns become rows

Code:


#include <stdio.h>

int main() {
    int a[2][3] = { {1, 2, 3},
                    {4, 5, 6} };
    int t[3][2];
    int i, j;

    for (i = 0; i < 2; i++)
        for (j = 0; j < 3; j++)
            t[j][i] = a[i][j];

    printf("Transpose:\n");
    for (i = 0; i < 3; i++) {
        for (j = 0; j < 2; j++)
            printf("%d ", t[i][j]);
        printf("\n");
    }

    return 0;
}

Call by Value

Concept

A copy of the actual argument is passed

Changes inside the function do not affect original variables

Example


#include <stdio.h>

void modify(int x) {
    x = x + 10;
    printf("Inside function: x = %d\n", x);
}

int main() {
    int a = 5;
    modify(a);
    printf("In main: a = %d\n", a);   // a unchanged
    return 0;
}

Call by Reference

Concept

Address of variable is passed

Changes inside function affect original variables

Example


#include <stdio.h>

void swap(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
}

int main() {
    int a = 10, b = 20;
    printf("Before: a = %d, b = %d\n", a, b);
    swap(&a, &b);
    printf("After: a = %d, b = %d\n", a, b);
    return 0;
}

Recursive Functions

Concept

A function that calls itself.

Important

Must have a base condition

Should reduce the problem size each call

Example (Factorial)


int fact(int n) {
    if (n == 0)
        return 1;          // base case
    else
        return n * fact(n - 1); // recursive call
}

Program to Find Power of a Number Using Recursion

Code:


#include <stdio.h>

int power(int base, int exp) {
    if (exp == 0)
        return 1;
    else
        return base * power(base, exp - 1);
}

int main() {
    int b, e;
    printf("Enter base and exponent: ");
    scanf("%d%d", &b, &e);

    printf("%d^%d = %d\n", b, e, power(b, e));
    return 0;
}

Problem 1 – Sum of Elements in an Array

Problem

Given n numbers in an array, find their sum.

Idea

Read all elements

Loop through the array

Keep adding to sum

Code


#include <stdio.h>

int main() {
    int n, i, sum = 0;
    printf("Enter n: ");
    scanf("%d", &n);

    int a[n];

    printf("Enter elements:\n");
    for (i = 0; i < n; i++)
        scanf("%d", &a[i]);

    for (i = 0; i < n; i++)
        sum += a[i];

    printf("Sum = %d\n", sum);
    return 0;
}

Problem 2 – Maximum Element

Problem

Find the largest number in an array.

Idea

Assume first element is maximum

Compare every value with current max

Update when larger value is found

Code


#include <stdio.h>

int main() {
    int n, i;
    printf("Enter n: ");
    scanf("%d", &n);

    int a[n];
    printf("Enter elements:\n");
    for (i = 0; i < n; i++)
        scanf("%d", &a[i]);

    int max = a[0];
    for (i = 1; i < n; i++) {
        if (a[i] > max)
            max = a[i];
    }

    printf("Max = %d\n", max);
    return 0;
}

Teaches

Condition checking

Loop logic

Pattern for many real interview problems

Union – Single Memory Location

A union is similar to a structure but more memory-efficient.

All members share the same memory location.

Only one member can contain a valid value at a time.

Useful when variables are used one at a time.

Helps in memory-constrained embedded systems.

Example:


union Data {
    int i;
    float f;
    char str[20];
};

Array of Structures – Multiple Records Stored

Stores multiple structure variables in a single array.

Useful for maintaining lists like students, employees, books, etc.

Allows easy traversal and searching.

Each element is a full structure containing all fields.

Supports sorting and file storage easily.

Syntax:


struct Student s[50];

Pointer Basics – Pointer Stores Address

A pointer is a variable that stores the address of another variable.

Provides indirect access to memory.

Basic syntax: int *p;

Enables dynamic memory allocation, linked lists, and arrays.

Used for efficient memory manipulation.

Declaring Pointers – Syntax

Declaration syntax: datatype *pointerName;

Example:

int *p; → pointer to integer

float *q; → pointer to float

Pointer must be assigned a valid address before using.

Use address-of operator & to store address: p = &x;

Dereferencing – Accessing Value Through Pointer

Dereferencing means accessing the value stored at the address in the pointer.

Use * operator to dereference.

Example:


int x = 10;
int *p = &x;
printf("%d", *p); // prints 10

Accessing Array Using Pointer – Pointer Arithmetic

Array name represents its base address.

Pointer can traverse array using arithmetic.

*(arr + i) is same as arr[i]

Incrementing pointer moves to the next element.

Efficient for scanning large arrays.

Example:


int *p = arr;
p++;

• data
